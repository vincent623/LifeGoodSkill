# Professional Problem Solver

Solve complex problems systematically using a four-tier framework for efficient solution development.

## Usage

```bash
# Describe your problem
/life-problem-solver

# Explain your situation and challenges when prompted
```

## The Four-Tier Framework

| Tier | Level | Focus |
|------|-------|-------|
| 1 | 调度级 (Dispatch) | Break down problem into expert domains, identify core challenges |
| 2 | 领域级 (Domain) | Connect context to relevant knowledge, ask targeted questions |
| 3 | 应用级 (Application) | Propose solutions, compare options, select best approach |
| 4 | 工具级 (Tools) | Provide action plans, testing methods, iteration feedback |

## What You Get

- **Problem Decomposition**: Clear understanding of core challenges
- **Expert Insights**: Knowledge-based analysis and recommendations
- **Solution Options**: Multiple approaches with pros/cons comparison
- **Action Plan**: Concrete steps with measurable outcomes

## Best For

- Complex decision making
- Business challenges
- Personal dilemmas
- Project problems
- Strategic planning
